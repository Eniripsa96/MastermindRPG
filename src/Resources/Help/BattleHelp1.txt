 -= Battle System Help Menu [Page 1] =-

Battle Symbols:
 - # = Cursor (hides tokens behind it)
 - X = Owned Token
 - O = Enemy Token

Battle Controls:
 - Move Cursor Up = Up Arrow
 - Move Crusor Down = Down Arrow
 - Move Cursor Left = Left Arrow
 - Move Cursor Right = Right Arrow
 - Inventory = I
 - Switch Action = Tab
 - Perform Action (Default is 'Place Token') = Space
 - Quit = Escape
 - Action Shortcuts = 1, 2, 3, 4, 5, 6, 7, 8, 9, or 0
     1 = Place Token         6 = Heal
     2 = Betray              7 = Rewind
     3 = Sacrifice           8 = Void
     4 = Double Strike       9 = Offering
     5 = Negotiate           0 = Dominate

'Esc' = Return ; 'Right Arrow' = Page 2
